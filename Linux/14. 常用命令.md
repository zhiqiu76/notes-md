- tar 命令

```
tar 
	-z	gzip进行解压或压缩，带.gz需要加，压缩出来.gz也需要加
	-x	解压
	-c	压缩
	-f	目标文件，压缩文件新命名或解压文件名
	-v	解压缩过程信息打印
解压命令：tar  -zvxf  xxxx.tar.gz

压缩命令：tar -zcf 压缩包命名 压缩目标
```
