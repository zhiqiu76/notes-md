- logging 模块

配置文件中有三个东西 日志器、处理器、格式器

==默认情况使用 root 日志器 默认使用 warning 级别==

```ini
[loggers]      # 定义日志器
keys=root

[logger_root]  # 实现日记器
#日志等级
level=DEBUG
# 指定处理器
handlers=hand01
# 日志器的名称
qualname=root
# 是否能够传播，propagete=0,表示输出日志，但消息不传递；
# propagate=1是输出日志，同时消息往更高级别的地方传递。
# root为最高级别。
propagate=0
# 指定编码格式
encoding=UTF-8

[handlers]         # 定义处理器
keys=hand01

[handler_hand01]   # 实现处理器
# 指定处理器类
class=FileHandler
# 日志级别
level=DEBUG
# 指定格式器
formatter=form01
# 处理器类参数, 第一个是文件path，第二个是文件打开方式a表示追加，w表示重写
args=('logs/file_log.log', 'w','utf8')   # 第三个参数指定 日志的编码

[formatters]    # 定义格式器
keys=form01

[formatter_form01]    # 实现格式器
# 格式化的模板
format=%(asctime)s %(filename)s[line:%(lineno)d] %(levelname)s %(message)s
# 指定日期格式
#datefmt=%a, %d %b %Y %H:%M:%S
datefmt=%Y-%m-%d %H:%M:%S, %a

```

```python
import logging

import logging.config

# 读取日志配置文件

logging.config.fileConfig('mysql.ini')

# 获取日志器
logger = logging.getLogger("root") # root是默认日志器

logger.debug("这是调试日志！")
logger.info("这是信息日志!")
logger.warning("这是警告！")
logger.error("这是严重错误！")
logger.critical("这是不饶恕的错误！")



```

- 把结果输出至控制台

```ini
[loggers]
keys=root,stream
[handlers]
keys=root_handler,stream
[formatters]
keys=formatter

[logger_root]
qualname=root
level=DEBUG
handlers=root_handler
propagate=0
encodeing=UTF-8

[logger_stream]
handlers=stream
qualname=stream
level=DEBUG
encoding=UTF-8
propagete=0

[handler_stream]
level=DEBUG
class=StreamHandler   # 处理类 为 StreamHandler
formatter=formatter
args=(sys.stdout,)    # sys.stdout 控制台 ，注意这里是一个元组

[handler_root_handler]
level=DEBUG
class=FileHandler
formatter=formatter
args=('logs/file.log','w','utf-8')

[formatter_formatter]
format=%(asctime)s %(filename)s %(levelname)s %(lineno)d %(name)s
datefmt=%Y-%m-%d %H:%M:%S, %a
```

==循环输出==

```ini
[loggers]
keys=root
[handlers]
keys=root,test
[formatters]
keys=formatter



[logger_root]
handlers=root,test
qualname=root
level=DEBUG
encoding=UTF-8
propagete=0

[handler_root]
level=DEBUG
class=logging.handlers.RotatingFileHandler         # 指定把日志记录到多个文件中去
formatter=formatter
args=('logs/rotating_file_log.log','a',1024,5,"utf-8")



[formatter_formatter]
format=%(asctime)s %(message)s
datefmt=%Y-%m-%d %H:%M:%S, %a

[handler_test]
level=DEBUG
class=StreamHandler
formatter=formatter
args=(sys.stdout,)
```
