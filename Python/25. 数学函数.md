- 内置函数

```python
# 绝对值
print(abs(-1))
# 从序列中获取最大值
print(max(1, 2))
# 从序列中获取最小值
print(min(2, 1))
# 幂运算
print(pow(3, 2))
# 返回浮点数x的四舍五入值，如给出n值，则代表舍入到小数点后的位数。
print(round(2.5653, 2))
# 返回有商和余数的元组
print(divmod(10, 3))
# 求和，iterable可迭代的对象， start指定相加参数，默认是0
print(sum((1, 2, 3, 4, 5), 1))

```

- math 模块

```python
import math
# 返回数字的上入整数，如math.ceil(4.1) 返回 5
print(math.ceil(1.12))
# 返回数字的下舍整数，如math.floor(4.9)返回 4
print(math.floor(1.56))
# 返回e的x次幂(ex),如math.exp(1) 返回2.718281828459045
print(math.exp(1))
# 返回数字的绝对值，返回浮点，如math.fabs(-10) 返回10.0
print(math.fabs(2))
# 返回以10为基数的x的对数，如math.log10(100)返回 2.0
print(math.log10(1))
# 返回x的整数部分与小数部分，两部分的数值符号与x相同，整数部分以浮点型表示。
print(math.modf(-5/3))
# 返回数字x的平方根
print(math.sqrt(16))

# math.pi就是π，math.sin(弧度)
print(math.sin(math.pi/2))

```

- 时区换算

```
from datetime import timezone, datetime, timedelta

time = datetime.utcnow() # 获取utc时间
print(time)

zone_time = time.replace(tzinfo=timezone.utc) 
print(zone_time)

utf_8 = timezone(timedelta(hours=8))

local_dt = zone_time.astimezone(utf_8)
print(local_dt)
```
