- 枚举就是常量的集合

==枚举元素的键是唯一的==

```python
from enum import Enum, unique

@unique
class Weekday(Enum):
    Sun = 0               # Sun的value被设定为0
    Mon = 1
    Tue = 2
    Wed = 3
    Thu = 4
    Fri = 5
    Sat = 6

for k,v in Weekday.__members__.items():
    print(k,v,v.value)
    
Weekday.sun              # Weekday.Sun
Weekday(0)               # Weekday.Sun
Weekday.sun.value        # 0
Weekday(0).value         # 0
    
Sun Weekday.Sun 0
Mon Weekday.Mon 1
Tue Weekday.Tue 2
Wed Weekday.Wed 3
Thu Weekday.Thu 4
Fri Weekday.Fri 5
Sat Weekday.Sat 6
```

> 创建枚举的第二个方法

```python
from enum import Enum

Month = Enum('Month', ('Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'))

for k,v in Month.__member__.items():
    print(k,v,v.value)
```
