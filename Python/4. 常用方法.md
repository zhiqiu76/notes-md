#### 常用方法

- random 随机数

```
import random
random.randint(1,5)  # 随机整数 [1,5] 闭区间  只有这个是闭区间
```
- isinstance(obj,class_or_tuple) 数据类型检查

```
#数据类型检查可以用内置函数
isinstance(1,int)
isinstance(1,(int,float))
```
- 进制转换
```
num = 5      #十进制表示    int(5)
num = 0b101  #二进制表示    bin(5)
num = 0o5    #八进制表示    oct(5)
num = 0x5    #十六进制表示  hex(5)

转换成十进制 int('100',2)
```

- 字符、码值转换

```
chr(97)   # 把码值转换成字符

ord('中') # 获取 unicode 码值
```
