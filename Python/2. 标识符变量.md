#### 格式化数据 format
==建议使用 format 格式化数据==
```
a=234
print('值：%s'%(a))
print('值: {}'.format(a))
其中，格式化整数和浮点数还可以指定是否补0和整数与小数的位数
print('%2d-%02d' % (3, 1))   #前面多出2位不补0，前面多出2位补0
print('%.2f' % 3.1415926)    #保留2位小数，默认保留6位

a=12345
b=12.1234
print('{:0>6}'.format(a))  # 012345   < 左对齐 > 右对齐
print('{:.2f}'.format(b))  # 12.12

```
