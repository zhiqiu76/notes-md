> 在 setting 加入

```python
LOGGING = {
    'version' : 1,
    'disable_existing_loggers' : False,
    'handlers' : {
        'console' : {
            'level' : 'DEBUG',
            'class' : 'logging.StreamHandler',
        },
    },
    'loggers' : {
        'django.db.backends' : {
            'handlers' : ['console'],
            'propagate' : True,
            'level' : 'DEBUG',
        },
    }
}
```
