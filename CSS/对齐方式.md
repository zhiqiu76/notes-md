- text-align

> 规定元素中 文本的对齐方式 

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        p{
            text-align: center;   /*居中对齐 水平对齐*/
            vertical-align: center; /*垂直对齐*/ 
            text-align: justify; /*使文本两端对齐*/
            
        }
    </style>
</head>

<body>
<p>test</p>

</body>
</html>
```
