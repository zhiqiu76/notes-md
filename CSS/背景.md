- 背景

==背景颜色==

> background-color: blue;

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .bgc{
            background-color: blue;
        }
    </style>
</head>

<body>
<div class="bgc">
    背景颜色
</div>
</body>
</html>
```

==背景图片==

> background-image: url();

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        div{
            background-image: url(images/1.jpg);  /*背景图片*/
            background-repeat: no-repeat;  /*背景图片是否重复 和 background-image 一起使用  默认是重复（水平和垂直） */
            height: 1000px;
        }
    </style>
</head>

<body>
<div>
    背景图片
</div>
</body>
</html>
```
