==样式继承只适用于元素的外观（文字、颜色、字体等），而元素在页面上的 布局样式则不会被继承。如果继承这种样式，就必须使用强制继承：inherit。==

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <p>这是<b>HTML5</b></p>
    <style type="text/css">
        p {
            border: 1px solid red;
        }

        b {
            border: inherit;             /*强制继承父类布局*/
        }
    </style>
</head>

<body>

<p>
    test
    <b>
        test
    </b>
</p>

</body>
</html>
```
