- 文本

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        p{
            color: red;   /*文本颜色*/
            text-align: center;   /*对齐方式*/
        }
    </style>
</head>

<body>
<p>test</p>
</body>
</html>
```

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        p{
            color: red;           /*文本颜色*/
            text-align: left;    /*文本对齐方式 只对块级元素有效*/
            text-decoration: underline;   /*文本修饰 underline，overline，line-through none*/
            text-indent: 2em;     /*首行缩进*/
        }
    </style>
</head>

<body>
<p>test</p>
<p ></p>
</body>
</html>
```

- 字体

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        p{
            font-family: 楷体,微软雅黑;
            font-size: 50px;      /*也可以使用 百分比 ，以父元素为标准*/
            font-style: italic;  /*字体风格*/
            font-weight: 100;  /*字体粗细 取值范围 100-900*/   

        }
    </style>
</head>

<body>
<p>你好</p>

</body>
</html>
```
